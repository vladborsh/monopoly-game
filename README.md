# Монополія UA Люкс — гра в браузері

Класична "Монополія" для двох гравців за одним екраном (hotseat).

## Запуск

```
npm install
npm run dev
```

Відкрий адресу, яку покаже термінал (зазвичай `http://localhost:5173`).

## Як грати

1. Хід переходить по черзі. На своєму ході натисни **«Кинути кубики»**.
2. Фішка автоматично рухається по колу на суму двох кубиків.
3. Якщо ти проходиш або зупиняєшся на полі **«Вперед»** — отримуєш зарплату 200 000.
4. Дії залежать від поля, на яке ти потрапив:
   - **Вільне поле (місто/компанія)** — можна купити або відмовитись.
   - **Поле, яким володіє суперник (без будинків)** — вибираєш: заплатити оренду або запропонувати викуп (120% від ціни). Власник погоджується або відхиляє пропозицію (відхилення = оренда все одно стягується).
   - **Поле з будинками** — оренда стягується автоматично.
   - **Податок** — списується автоматично (на полі "Податок на нерухомість" сума росте залежно від кількості твоїх будинків на всій дошці).
   - **Шанс / Казна** — тягнеш картку, ефект застосовується одразу.
   - **Казино** — можна зіграти на ставку 10 000 (шанс виграти множник) або пропустити.
   - **В'язниця** — щоб вийти: викинути дублі, заплатити заставу, або використати картку "звільнення з в'язниці". Після 3 невдалих спроб застава списується автоматично.
5. Коли скупиш усі поля одного кольору — можна будувати будинки (кнопка з'являється на панелі дій), поки не заповнено 3 будинки на полі. Будинки збільшують оренду.
6. Хід завершується кнопкою **«Завершити хід»**.
7. Якщо в когось закінчуються гроші — гравець вибуває, а всі його володіння повертаються банку.
8. Гра закінчується, коли залишається один гравець.

## Панель керування

- Праворуч — інформація про гравців (гроші, позиція, статус), кнопки дій та лог подій.
- Гра автоматично зберігається в браузері (localStorage) — можна закрити вкладку і продовжити пізніше.
- Кнопка **«Нова гра»** починає гру заново з тими ж гравцями; в налаштуваннях рестарту можна змінити кількість гравців і стартовий капітал.
